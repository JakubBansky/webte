Tile Painter

Play game

Instructions


Paint all tiles inside a house, once you finish all 5 levels, you can either quit or play again.
PC
On the computer, you control the game using the keys (up, down, left, right) on the keyboard. After pressing the up button, the current position moves, and it colors the tiles that can be colored above the current position. After pressing the down button, the current position moves, and it colors the tiles that can be colored below the current position. After pressing the left button, the current position moves, and it colors the tiles that can be colored to the left of the current position. After pressing the right button, the current position moves, and it colors the tiles that can be colored to the right of the current position.
Mobile
On an android mobile device, you use the built-in gyroscope to control the game by tilting the phone on the sides. Please use portrait mode (camera of a phone is on top). After tilting the phone to the left, the current position moves, and it colors the tiles that can be colored to the left of the current position. After tilting the phone to the right, the current position moves, and it colors the tiles that can be colored to the right of the current position. After tilting the phone up, the current position moves, and it colors the squares that can be colored above the current position. After tilting the phone down, the current position moves, and it colors the squares that can be colored below the current position.
The current position is marked with a white color in each level. Orange color is coloring and also the color of the starting position. Tiles marked with wood are barriers and cannot be colored. The task of each level is to color all the tiles designated for coloring.
Now you are all set, have fun.
